## 🚀 Phase 7 Architecture

### Session 1: Security Hardening (2 hours)

**Files to Create:**
- `lib/validation.ts` - Input sanitization
- `middleware/security.ts` - Security headers
- Update ALL `/app/api/*/route.ts` - Add validation

**Tasks:**
1. Add Zod validation schemas
2. Input sanitization (prevent XSS)
3. SQL injection audit (verify all queries use `?`)
4. Rate limiting review (verify: 10/min signals, 30/min messages)
5. CORS lockdown (change `*` to specific origins)
6. Request size limits (10MB max)
7. API key rotation endpoint

**Validation Rules:**
- Agent name: 3-50 chars, alphanumeric
- Signal content: 1-1000 chars
- Wallet amount: positive number
- Voting mode: enum ['majority', 'supermajority', 'consensus']
