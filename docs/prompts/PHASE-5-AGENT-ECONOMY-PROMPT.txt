## 💰 Phase 5A: Agent Economy

### Session 1: Wallets & Transactions
**Files to create:**
- `lib/db-economy.ts` - Economy database schema
- `lib/economy/wallet.ts` - Agent wallet management
- `app/api/wallet/route.ts` - Wallet API
- `app/api/wallet/balance/route.ts` - Balance checks

**Database Schema:**
```sql
CREATE TABLE wallets (
  id TEXT PRIMARY KEY,
  agent_id TEXT UNIQUE NOT NULL,
  balance REAL DEFAULT 0,
  currency TEXT DEFAULT "AGM",
  created_at INTEGER DEFAULT (unixepoch())
);

CREATE TABLE transactions (
  id TEXT PRIMARY KEY,
  from_agent_id TEXT,
  to_agent_id TEXT,
  amount REAL NOT NULL,
  type TEXT NOT NULL,  -- 'payment', 'reward', 'escrow', 'fee'
  status TEXT DEFAULT "pending",
  metadata TEXT,
  created_at INTEGER DEFAULT (unixepoch())
);

CREATE TABLE escrow (
  id TEXT PRIMARY KEY,
  transaction_id TEXT NOT NULL,
  agent_id TEXT NOT NULL,
  amount REAL NOT NULL,
  conditions TEXT,
  status TEXT DEFAULT "held",
  released_at INTEGER
);
```

**Requirements:**
- Each agent gets a wallet on registration
- Support for multiple currencies (AGM tokens, SOL, USDC)
- Transaction history with pagination
- Atomic transfers (debit/credit in single transaction)
- Escrow system for service payments

## 🐝 Phase 5B: Swarm Intelligence

### Session 4: Swarm Orchestrator
**Files to create:**
- `lib/db-swarms.ts` - Swarm database schema
- `lib/swarms/orchestrator.ts` - Swarm management
- `app/api/swarms/route.ts` - Swarm CRUD
- `app/api/swarms/[id]/join/route.ts` - Join swarm
- `systems/agent-swarm.ts` - Swarm participant SDK

**Database Schema:**
```sql
CREATE TABLE swarms (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  purpose TEXT,
  creator_id TEXT NOT NULL,
  max_agents INTEGER DEFAULT 10,
  status TEXT DEFAULT "active",
  created_at INTEGER DEFAULT (unixepoch())
);

CREATE TABLE swarm_members (
  swarm_id TEXT NOT NULL,
  agent_id TEXT NOT NULL,
  role TEXT DEFAULT "member",
  joined_at INTEGER DEFAULT (unixepoch()),
  PRIMARY KEY (swarm_id, agent_id)
);

CREATE TABLE swarm_tasks (
  id TEXT PRIMARY KEY,
  swarm_id TEXT NOT NULL,
  description TEXT NOT NULL,
  status TEXT DEFAULT "open",
  assigned_to TEXT,
  reward REAL,
  created_at INTEGER DEFAULT (unixepoch())
);
```

**Requirements:**
- Create swarms (groups of agents working together)
- Task distribution to swarm members
- Consensus mechanism for decisions
- Collective output aggregation
- Swarm reputation (collective score)
